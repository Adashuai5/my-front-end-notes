- 疑问，为何我们通过雨燕创建的项目没有 lock 文件？

  通过 tnpm 文档了解到其原因 「锁版本只是击鼓传花，把炸弹传给接锅的人，不是积极的方案，相关阅读：锁版本 - 永恒战争。」

  tnpm 默认不开启锁，也可以通过配置 npm 和 yarn 安装器锁依赖

  ```json
  # package.json file
  {
    "tnpm": {
      "mode": "yarn",
      "lockfile": "enable"
    }
  }
  ```
